---
title: "Remove users from a limit"
description: "Remove users from a shared fund (limit)."
api: "POST https://api.ramp.com/developer/v1/limits/{spend_limit_id}/delete-users"
---

**Required scope:** `limits:write`

## Path Parameters

<ParamField path="spend_limit_id" type="string" required>The limit UUID.</ParamField>

## Body

<ParamField body="user_ids" type="array">
  Array of user UUIDs to remove from this shared limit.
</ParamField>

## Response

Returns the updated limit object.
