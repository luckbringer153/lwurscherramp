---
title: "List bills"
description: "Retrieve a paginated list of bills."
api: "GET https://api.ramp.com/developer/v1/bills"
---

**Required scope:** `bills:read`

## Query Parameters

<ParamField query="sync_ready" type="boolean">
  Filter to bills that are ready to sync to accounting system.
</ParamField>

<ParamField query="payment_sync_ready" type="boolean">
  Filter to bills that are invoice-synced but not yet payment-synced.
</ParamField>

<ParamField query="sync_status" type="string">
  Filter by sync status. One of: `NOT_SYNCED`, `BILL_SYNCED`, `BILL_AND_PAYMENT_SYNCED`.
</ParamField>

<ParamField query="start" type="string">
  Pagination cursor. Use the value of `page.next` from a previous response.
</ParamField>

<ParamField query="page_size" type="integer">
  Number of results per page (default: 100, max: 100).
</ParamField>
