---
title: "Update Inventory Item Option"
description: "Update an inventory item option."
api: "PATCH https://api.ramp.com/developer/v1/accounting/inventory-item/options/{option_id}"
---

## Path Parameters

<ParamField path="option_id" type="string" required>
  The ID of the option to update.
</ParamField>

## Request

<ParamField body="name" type="string">
  Updated name.
</ParamField>
<ParamField body="reactivate" type="boolean">
  Set to `true` to reactivate a deactivated option.
</ParamField>

## Response

<ResponseField name="accounting_connection_id" type="string" required />
<ResponseField name="id" type="string" required />
<ResponseField name="ramp_id" type="string" required />
<ResponseField name="name" type="string" required />
<ResponseField name="is_active" type="boolean" required />
