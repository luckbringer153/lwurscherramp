---
title: "Upload Inventory Item Options"
description: "Upload options for the inventory item accounting field."
api: "POST https://api.ramp.com/developer/v1/accounting/inventory-item/options"
---

There must be an active inventory item accounting field for the accounting connection before uploading options.

## Request

<ParamField body="options" type="array" required>
  <Expandable title="options[]">
    <ParamField body="id" type="string" required>
      Your ERP's unique identifier for this option.
    </ParamField>
    <ParamField body="name" type="string" required>
      Display name of the option.
    </ParamField>
  </Expandable>
</ParamField>
