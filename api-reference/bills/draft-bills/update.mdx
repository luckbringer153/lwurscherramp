**Required scope:** `bills:write`

## Path Parameters

<ParamField path="draft_bill_id" type="string" required>
  The Ramp UUID of the draft bill to update.
</ParamField>

<ParamField body="vendor_id" type="string">

</ParamField>

<ParamField body="due_at" type="string">
  Updated due date in ISO 8601 format.
</ParamField>

<ParamField body="issued_at" type="string">
  Updated invoice issue date.
</ParamField>

<ParamField body="invoice_number" type="string">
  Updated invoice number.
</ParamField>

<ParamField body="entity_id" type="string">
  Updated business entity UUID.
</ParamField>

<ParamField body="invoice_currency" type="string">
  Updated invoice currency (ISO 4217).
</ParamField>

<ParamField body="line_items" type="array">
  Replaces all line items. Each item: `amount`, `memo`, `accounting_field_selections`.
</ParamField>

<ParamField body="inventory_line_items" type="array">
  Replaces all inventory line items. Each item: `quantity`, `unit_price`, `memo`, `accounting_field_selections`.
</ParamField>

<ParamField body="accounting_field_selections" type="array">
  Updated bill-level accounting coding.
</ParamField>

<ParamField body="payment_method" type="string">

<ParamField body="vendor_contact_id" type="string">

</ParamField>

<ParamField body="use_default_payment_method" type="boolean">

<ParamField body="purchase_order_ids" type="array">

<ParamField body="memo" type="string">

Returns the updated draft bill object. See [Fetch a draft bill](/api-reference/bills/draft-bills/get) for the response schema.
