---
title: "Reactivate Accounting Connection"
description: "Reactivate a previously disconnected accounting connection."
api: "POST https://api.ramp.com/developer/v1/accounting/connection/{connection_id}/reactivate"
---

Reactivates a previously disconnected accounting connection by changing its status back to `linked`. This preserves all previous accounting field configurations and settings.

<Note>
  The business must not have any other active accounting connections.
</Note>

## Path Parameters

<ParamField path="connection_id" type="string" required>
  The ID of the accounting connection to reactivate.
</ParamField>

## Response

<ResponseField name="id" type="string" required />
<ResponseField name="connection_type" type="string" required />
<ResponseField name="remote_provider_name" type="string" required />
<ResponseField name="status" type="string" required />
<ResponseField name="is_active" type="boolean" required />
<ResponseField name="is_ready_to_migrate" type="boolean" required />
<ResponseField name="created_at" type="string (date)" required />
<ResponseField name="last_linked_at" type="string (date)" required />
<ResponseField name="settings" type="object">
  <Expandable title="settings">
    <ResponseField name="reimbursement_sync_button_enabled" type="boolean" />
    <ResponseField name="transaction_sync_button_enabled" type="boolean" />
    <ResponseField name="transaction_accounting_vendor_creation_on_sync_enabled" type="boolean" />
    <ResponseField name="vendor_credits_enabled" type="boolean" />
  </Expandable>
</ResponseField>
