---
title: "Update GL Account"
description: "Update the name or code of a general ledger account."
api: "PATCH https://api.ramp.com/developer/v1/accounting/accounts/{gl_account_id}"
---

## Path Parameters

<ParamField path="gl_account_id" type="string" required>
  The Ramp ID of the GL account to update.
</ParamField>

## Request

<ParamField body="name" type="string">
  Updated display name.
</ParamField>
<ParamField body="code" type="string">
  Updated account code.
</ParamField>
<ParamField body="reactivate" type="boolean">
  Set to `true` to reactivate a previously deactivated account.
</ParamField>
