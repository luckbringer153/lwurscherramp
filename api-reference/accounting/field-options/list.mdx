## Query Parameters

<ParamField query="field_id" type="string">
  Filter options by accounting field ID.
</ParamField>
<ParamField query="visibility" type="string">
  Filter by visibility: `HIDDEN` or `VISIBLE`.
</ParamField>
<ParamField query="is_active" type="boolean">
  Filter by active status.
</ParamField>
