---
title: "Fetch a cashback payment"
description: "Retrieve a single cashback payment by ID."
api: "GET https://api.ramp.com/developer/v1/cashbacks/{cashback_id}"
---

**Required scope:** `cashbacks:read`

## Path Parameters

<ParamField path="cashback_id" type="string" required>
  The Ramp UUID of the cashback payment.
</ParamField>

## Response

<ResponseField name="id" type="string">Cashback payment UUID.</ResponseField>
<ResponseField name="amount" type="object">
  Cashback amount (CurrencyAmount with `amount`, `currency_code`, `minor_unit_conversion_rate`).
</ResponseField>
<ResponseField name="sync_status" type="string">`NOT_SYNC_READY` | `SYNC_READY` | `SYNCED`</ResponseField>
<ResponseField name="entity_id" type="string">Business entity UUID.</ResponseField>
<ResponseField name="created_at" type="string">Payment timestamp.</ResponseField>
