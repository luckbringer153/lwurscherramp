---
title: "Create Inventory Item Field"
description: "Create the inventory item accounting field for the current accounting connection."
api: "POST https://api.ramp.com/developer/v1/accounting/inventory-item"
---

There can only be one active inventory item accounting field per accounting connection.

## Request

<ParamField body="name" type="string" required>
  Name of the inventory item field.
</ParamField>
<ParamField body="display_name" type="string">
  Optional display name shown in the Ramp UI.
</ParamField>
