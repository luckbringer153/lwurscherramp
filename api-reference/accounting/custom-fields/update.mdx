## Path Parameters

<ParamField path="field_id" type="string" required>

</ParamField>

## Request

<ParamField body="name" type="string">

</ParamField>
<ParamField body="display_name" type="string">

</ParamField>
<ParamField body="is_splittable" type="boolean">
  Whether this field can be split across multiple values.
</ParamField>
